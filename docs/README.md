# Amazon Clone - Project Documentation

Welcome to the Amazon Clone project! This project is a static HTML and CSS recreation of the Amazon website. It is designed to help you understand the core concepts of web development without the complexity of modern frameworks.

## Project Structure

- `index.html`: The main homepage content.
- `search.html`: Example search results page.
- `product.html`: Example product detail page.
- `cart.html`: Shopping cart page.
- `signin.html`: Sign-in page.
- `checkout.html`: Checkout summary page.
- `css/`: Contains all the styling files.
    - `style.css`: Global styles.
    - `header.css`: Styles for the navigation bar.
    - `footer.css`: Styles for the footer.
    - `home.css`, `product.css`, etc.: Page-specific styles.

## How to Run
Simply open `index.html` in your web browser. You can click on links to navigate between the different pages.
